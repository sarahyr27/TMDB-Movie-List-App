# TMDB Movie Web App

This app was developed as a part of a university project for the Digital Interaction Techniques and Multimedia degree at Universitat Oberta de Catalunya. Built on JavaScript, HTML, and CSS, this was my first experience with REST APIs and basic debugging, which was actually the main focus of the project.

---

## Features:

- **User management on the browser's local storage:** To use the app, it’s mandatory to create an account and log in.
- **Browsing movies:** With the integration of the TMDB API, the user has access to a massive and updated movie catalogue, and has the possibility to filter the results by genre.
- **Managing lists:** Another of the app’s functions allows you to add or delete movies to the three lists available by default (favorites, already watched, and watchlist).

---

## Technologies

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **API Integration:** TMDB API for fetching movie data
- **Data Storage:** localStorage for persistent user data
- **Tools & Libraries:**
  - Chrome / Firefox DevTools for debugging
  - VS Code for development
  - Git/GitHub for version control
